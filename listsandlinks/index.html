How to put a Webpage on the Prodution Server

  1. Make a .html file inside your directory
  2. Save and validate your content
  3. Check and correct all the errors if there are any
  4. After completing the first three steps, open terminal
  5. Go into your directory folder or the folder where the .html file is located
  6. Type in- sftp (name&studentnumber)@cmis-hsteacher-
  7. Go into the public.html folder
  8. Put the .html folder into your current folder by typing - put (foldername).html-
  9. To check the .html file on the production server type - cmis-hsteacher/~(name&studentnumber)/ -
